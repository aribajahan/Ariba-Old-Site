How to customize the current theme:

Octopress themes are based on Jekyll and Shopify template language Liquid (both written in Ruby). Learning some CSS and HTML will help you move forward.

Themes are located in the .themes/ folder and when you rake install ["theme_name"] it installs it to the folders:sass and source. This essentially means you have 3 copies of an installed theme: 1. public repo 2. in octopress 3. in .themes/
